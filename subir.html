Subir diseñadoras a la web

Si estás interesado en colaborar con este proyecto subiendo la información de una diseñadora esta es la ficha que debes rellenar. A continuación este material será subido a la web para que todo el mundo pueda conocer a esta autora y utilizarla como referencia. El objetivo de este formulario es recoger datos para iniciar una red colaborativa que sirva para buscar y compartir referentes femeninos en el campo del diseño. Para esto te pedimos que elijas una diseñadora y rellenes las preguntas que te haremos a continuación. No te cortes en ir a buscar a internet o a donde sea los datos que desconozcas. La motivación principal es recopilar datos para que otros usuarios puedan leer sobre diseñadoras que antes no conocían. Si quieres introducir más de una diseñadora, solo tienes que rellenar el formulario para cada una de ellas.

Bibliografía de la diseñadora

Incluye alguna foto de ella y de sus obras, copiando y pegando su link a continuación. El link debe hacer referencia al recurso, no a la página que lo contiene. Se consigue al inspeccionar sobre la foto, copiando el link al que hace referencia. Tendrá una forma parecida a ésta: "//c1.staticflickr.com/1/914/42350277715_048fbce61e_z.jpg" El tamaño recomendado para las fotos es 1280 x 1280.
